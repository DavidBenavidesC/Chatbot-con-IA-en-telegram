# Chat IA en Telegram con Node.js, n8n, Docker y Google Gemini

Este repositorio acompaña un video tutorial en el que se muestra paso a paso cómo crear y configurar un chatbot con inteligencia artificial para Telegram utilizando Node.js, n8n, Docker, Ngrok y Google Gemini.

## Descripción

A lo largo del tutorial se explica el proceso completo para desplegar un asistente inteligente en Telegram, desde la instalación de las herramientas necesarias hasta la configuración del flujo de trabajo y la integración con un modelo de IA.

## Características

- Instalación y configuración de Node.js.
- Configuración y ejecución de n8n.
- Automatización del inicio de n8n.
- Exposición de servicios locales mediante Ngrok.
- Despliegue utilizando Docker.
- Creación y configuración de un bot con BotFather.
- Importación y personalización de un flujo de trabajo de IA.
- Integración con Google Gemini.
- Pruebas y puesta en funcionamiento del asistente.

## Tecnologías utilizadas

- Node.js
- n8n
- Docker
- Ngrok
- Telegram Bot API
- Google Gemini API

## Objetivo

El objetivo de este proyecto es servir como una guía práctica para cualquier persona interesada en desarrollar asistentes inteligentes en Telegram mediante herramientas accesibles y fácilmente personalizables.

## Resultado

Al finalizar el tutorial, se obtiene un chatbot funcional capaz de responder a los usuarios mediante los modelos de inteligencia artificial de Google Gemini.
